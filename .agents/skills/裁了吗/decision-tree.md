# 决策分流系统（Decision Tree）

本文件用于把用户问题快速分流到可执行流程。  
目标：先分类，再细分，再调用资源，再按统一 8 段结构输出。

## 使用规则

1. 先做第一层判断（问题类型）
2. 再做第二层判断（同一类型下的关键状态）
3. 按分支加载资源文件
4. 输出时统一使用 8 段结构

---

## 第一层判断（问题分类）

当用户输入后，优先判断命中以下哪一类：

1. 刚被裁 / HR / 签字 / 补偿 -> **裁员流程分支**
2. 情绪崩溃 / 焦虑 / 自责 -> **情绪稳定分支**
3. 不知道未来方向 -> **方向决策分支**
4. 想转行 -> **转行分析分支**
5. 想单干 / 副业 -> **单干副业分支**
6. 想要计划 -> **行动计划分支**

如果同时命中多个类别，优先级如下：

1. 情绪稳定分支
2. 裁员流程分支
3. 方向决策 / 转行分析 / 单干副业分支
4. 行动计划分支

---

## 分支 A：裁员流程分支

### 第二层判断

1. 是否已经签字？
- 未签字：优先拖时间 + 要书面 + 核条款
- 已签字：优先做文本复盘 + 风险评估 + 后续路径

2. 是否正在谈补偿？
- 是：聚焦补偿计算口径、发放时间、税费与附加条款
- 否：先确认解除性质、时间节点、书面版本

3. 是否涉及竞业？
- 是：优先核对范围、期限、补偿、违约责任
- 否：聚焦补偿、背调、离职节奏

4. 是否考虑仲裁？
- 是：先做争议点清单与证据清单，再建议专业咨询
- 否：优先走协商与可逆方案

### 调用资源

- `resources/被裁当天.md`
- `resources/HR谈判.md`
- `resources/和直属领导谈.md`
- `resources/离职与背调.md`
- `resources/竞业协议.md`
- `resources/劳动仲裁基础.md`
- `resources/被裁当天-checklist.md`

### 输出重点

- 先稳住 + 关键事实确认
- 今天优先做的 3-5 件事
- HR/领导可直接发送话术
- 签字前风险提醒

---

## 分支 B：情绪稳定分支

### 第二层判断

1. 当前状态属于哪种？
- 崩溃失控
- 持续焦虑
- 强烈自责/羞耻

2. 是否影响基本功能？
- 已影响睡眠/进食/工作：先恢复执行能力
- 未明显影响：进入低负荷行动节奏

3. 是否出现高风险信号？
- 有明显极端表达：立即建议专业支持
- 无：按 24-72 小时稳定方案推进

### 调用资源

- `resources/情绪稳定.md`
- `resources/被裁当天-checklist.md`
- `resources/一周-checklist.md`

### 输出重点

- 30 分钟稳定动作
- 今晚只做 1-2 个现实动作
- 72 小时内禁止事项与节奏建议

---

## 分支 C：方向决策分支

### 第二层判断

1. 主要困惑是什么？
- 继续做开发还是换方向
- 年龄焦虑导致方向摇摆
- 多路径并行但无优先级

2. 当前约束是什么？
- 现金流压力
- 家庭压力
- 地域/岗位机会密度

3. 是否已有备选路径？
- 有：建立主备切换阈值
- 无：先生成主路径 + 备路径

### 调用资源

- `resources/未来方向分析.md`
- `resources/找工作-checklist.md`
- `resources/7天计划.md`
- `resources/30天计划.md`

### 输出重点

- 主路径 + 备路径
- 14 天验证指标
- 7/30/90 天节奏

---

## 分支 D：转行分析分支

### 第二层判断

1. 转行方向是否明确？
- 明确：评估入场路径与成本
- 不明确：先做方向收敛

2. 是否可先低风险验证？
- 可：先副路径验证，不冲动裸辞
- 不可：先保收入再转向

3. 是否有作品/证据？
- 有：进入投递与访谈
- 无：先做最小作品与能力证明

### 调用资源

- `resources/转行分析.md`
- `resources/转行准备-checklist.md`
- `resources/30天计划.md`

### 输出重点

- 可行性评估
- 最小试错动作
- 回撤条件与风险边界

---

## 分支 E：单干副业分支

### 第二层判断

1. 是副业试水还是全职单干？
- 副业试水：先验证需求与付费信号
- 全职单干：先核现金流与成交路径

2. 是否有明确服务与客户？
- 有：进入触达与试点
- 无：先定义一句话服务

3. 是否达到继续条件？
- 有付费信号：可加码
- 无付费信号：降级为副路径

### 调用资源

- `resources/单干与副业.md`
- `resources/单干试水-checklist.md`
- `resources/一个月-checklist.md`

### 输出重点

- 30 天试水计划
- 最低收入阈值
- 继续/暂停条件

---

## 分支 F：行动计划分支

### 第二层判断

1. 用户需要哪个时间尺度？
- 今天
- 本周（7天）
- 一个月（30天）
- 三个月（90天）

2. 当前主问题属于哪条主分支？
- 裁员流程
- 情绪稳定
- 方向决策/转行/单干

3. 计划是否可执行？
- 任务过多：先减到关键 3-5 条
- 任务可执行：按周复盘推进

### 调用资源

- `resources/7天计划.md`
- `resources/30天计划.md`
- `resources/90天计划.md`
- `resources/一周-checklist.md`
- `resources/一个月-checklist.md`

### 输出重点

- 今天/本周可执行清单
- 每周复盘点
- 7/30/90 天目标对齐

---

## 统一输出约束（所有分支）

无论进入哪个分支，最终输出都按以下 8 段：

1. 先稳住（情绪稳定）
2. 当前局面判断
3. 现在最重要的事
4. 行动清单（今天 / 本周）
5. 可以说的话（HR / 领导 / 家人 / 面试）
6. 风险提醒
7. 接下来时间线（7天 / 30天 / 90天）
8. 行动收束总结
