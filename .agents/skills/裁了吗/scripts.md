# 话术模板库（可直接复制）

使用建议：
- 先发短版本，必要时再补充细节。
- 关键点尽量要求书面确认。
- 占位符按实际替换：`[姓名] [日期] [岗位] [金额] [公司]`。

## 1) 跟 HR 拖时间

```text
你好 [HR姓名]，
我希望以理性、专业的方式完成这次沟通。
当前条款我需要进一步核对，请将签署截止时间延至 [日期]。
我会在收到完整信息后尽快给出反馈。谢谢。
```

## 2) 跟 HR 要书面方案

```text
你好 [HR姓名]，
为了避免理解偏差，请将本次沟通结论以书面形式确认，包含：
1）解除/离职性质；
2）补偿计算方式；
3）发放时间；
4）社保公积金与年假结算安排；
5）签署截止日期。
我会以书面版本作为后续处理依据。
```

## 3) 跟 HR 谈补偿

```text
你好 [HR姓名]，
关于补偿方案，我需要确认以下关键点：
1）计算基数采用哪段期间平均工资；
2）补偿月份构成（含是否为代通知金）；
3）税费口径；
4）具体发放日期与到账方式。
请帮我书面确认，谢谢。
```

## 4) 跟 HR 拒绝主动离职

```text
你好 [HR姓名]，
我目前不接受“主动离职”处理方式。
如公司提出岗位调整或解除安排，请按正式流程和书面方案沟通。
我会基于正式文本进行审阅与回复。
```

## 5) 跟 HR 谈竞业补偿

```text
你好 [HR姓名]，
关于竞业条款，请书面明确：
1）适用范围（行业、岗位、地域）；
2）限制期限；
3）竞业补偿标准与发放时间；
4）违约责任边界。
在上述信息明确前，我暂不做最终确认。
```

## 6) 跟领导要推荐 / 背调

```text
领导好，我正在推进下一步求职，想请您帮我两件事：
1）如方便，作为我的背调联系人；
2）如有匹配岗位，帮我做一次内推或转介绍。
我会先整理一页项目成果要点，尽量不占用您时间。谢谢您。
```

## 7) 跟家里人解释被裁

```text
我这边遇到岗位调整，短期会有压力，但我已经在按计划处理。
这周我先处理协议和现金流，下周集中推进求职和内推。
你们先不用太担心，我会每周同步进展。
```

## 8) 找前同事内推

```text
嗨 [同事名]，最近还好吗？
我目前在看 [岗位方向] 的机会，如果你们团队有匹配岗位，方便帮我看下是否适合内推吗？
我可以把最新简历和 3 个项目成果点发你，感谢。
```

## 9) 面试解释离职原因

```text
我所在团队经历组织调整，我的岗位被优化。
这段时间我系统复盘了过往项目，并补齐了 [能力项]。
我现在重点看 [岗位方向]，希望加入一个稳定、重视长期建设的团队。
```

## 10) 找工作被问空窗期

```text
这段空窗期我主要做了三件事：
1）系统复盘并优化了项目叙事与简历；
2）补齐了 [技术/业务能力]；
3）持续参与面试与行业沟通。
这段时间让我在岗位匹配和产出节奏上更清晰。
```

## 11) 拒绝不合理 offer

```text
感谢贵司的认可和 offer。
我认真评估后，当前条件与我的预期存在较大差距（岗位职责/薪资结构/工作安排）。
这次我先不接受该 offer。
如果后续条件有调整，欢迎再沟通。谢谢理解。
```

## 12) 谈薪资

```text
感谢贵司的机会。
结合岗位职责、我的相关经验以及市场区间，我的期望是 [金额区间]。
如果总包结构可优化（如基础薪资/奖金/补贴），我愿意继续沟通并尽快确认。
```

## 13) 请求延迟离职日期

```text
你好 [HR姓名]，
考虑到交接安排与个人事务处理，我申请将离职生效日期调整至 [日期]。
我会在此期间配合完成必要交接，确保工作平稳过渡。
请确认是否可行，谢谢。
```

## 14) 补充模板：会后纪要确认

```text
你好 [HR姓名]，
感谢今天沟通。为避免偏差，我整理会议要点如下：
1）[要点1]
2）[要点2]
3）[要点3]
如有不一致，请在 [时间] 前书面更正。
```

## 15) 补充模板：拒绝违法/破坏行为

```text
这类做法我不能协助，包括删数据、泄密、带走代码或破坏设备。
如果目标是保护自身权益，我可以帮你走合规路径：
证据整理、书面沟通、风险评估和行动计划。
```
