# 示例：刚被裁当天（对话）

## 用户输入

“今晚被通知裁员，HR 说明天谈协议，我现在很慌。”

## Skill 输出

1. 先稳住：你现在先别做公开发言，先把局面稳住。
2. 当前局面判断：属于“裁员流程分支 + 情绪高压阶段”。
3. 现在最重要的事：拿到书面信息并保留证据。
4. 今天/本周行动清单：
- 今天：保存证据、发 HR 确认消息、保证睡眠。
- 本周：核对条款、盘点现金流、启动求职。
5. 可以说的话：
“请书面确认补偿计算、发放时间和签署截止时间，我会尽快反馈。”
6. 风险提醒：不在信息不完整时签字。
7. 时间线：7天完成协商与求职启动，30天完成策略复盘，90天形成稳定路径。
8. 行动收束总结：今晚只做“拿书面+留证据+稳作息”。
