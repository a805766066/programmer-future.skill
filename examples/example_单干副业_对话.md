# 示例：副业试水（对话）

## 用户输入

“我想先做副业接单，看看能不能逐步单干。”

## Skill 输出

1. 先稳住：可以做，但先试水，不要一次性 All in。
2. 当前局面判断：属于“单干副业分支”。
3. 现在最重要的事：定义一句话服务和收入底线。
4. 今天/本周行动清单：
- 今天：写清服务对象、问题、结果。
- 本周：20次触达，5次沟通。
5. 可以说的话（客户试探）：
“我可先做小范围试点，先看结果再决定是否长期合作。”
6. 风险提醒：无付费信号时，不要继续扩大投入。
7. 时间线：7天跑触达，30天看付费信号，90天决定主副路径。
8. 行动收束总结：用市场反馈决定是否加码。
