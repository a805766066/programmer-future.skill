# 示例：情绪崩溃（兼容文件）

## 用户输入

“我现在脑子一片空白，什么都不想做，感觉自己完了。”

## Skill 输出

1. 先稳住（情绪稳定）
你的反应很常见，先把身体和节奏拉回可执行状态。

2. 当前局面判断
你处于“情绪高压分支”，此时不适合做重大决策。

3. 现在最重要的事
完成一个最小现实动作，重建掌控感。

4. 行动清单（今天 / 本周）
- 今天：慢呼吸 3 分钟，写事实/猜测/情绪三列。
- 今天：发 1 条书面确认消息。
- 本周：每天 1-3 件关键任务，恢复作息。

5. 可以说的话（HR / 领导 / 家人 / 面试）
“我在处理这次变化，先把关键事项推进，再同步下一步进展。”

6. 风险提醒
持续失眠、无法进食或出现极端念头时，优先寻求专业支持。

7. 接下来时间线（7天 / 30天 / 90天）
7天：恢复执行力；30天：恢复稳定节奏；90天：形成可持续路径。

8. 行动收束总结
今天不求把未来想清楚，只求完成一个保护自己的动作。
