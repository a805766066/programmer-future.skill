# 示例：是否降薪换工作（对话）

## 用户输入

“我现在拿不到同级薪资的 offer，要不要先降薪入职？”

## Skill 输出

1. 先稳住（情绪稳定）
先不要把“降薪”当失败，先把它当阶段策略来评估。

2. 当前局面判断
你处于“方向决策分支”，核心是现金流压力与长期成长的平衡。

3. 现在最重要的事
判断这次降薪是否能换来更高的稳定性和成长性。

4. 行动清单（今天 / 本周）
- 今天：列出最低可接受收入底线。
- 今天：比较该岗位的成长空间、团队质量、业务稳定性。
- 本周：并行保留 1-2 条更优机会，不停止面试。

5. 可以说的话（HR / 领导 / 家人 / 面试）
“如果总包暂时难到我的目标区间，我可以接受阶段性方案，但希望在职责、评估周期和调薪机制上有明确约定。”

6. 风险提醒
只降薪不换成长空间，容易陷入二次跳槽。

7. 接下来时间线（7天 / 30天 / 90天）
7天完成岗位价值评估；30天评估入职后匹配度；90天复盘是否继续或切换。

8. 行动收束总结
能降薪，但不能盲降，先保底再保成长。
