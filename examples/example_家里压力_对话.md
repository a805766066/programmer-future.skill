# 示例：家里压力（对话）

## 用户输入

“家里一直催我赶紧稳定，我压力特别大。”

## Skill 输出

1. 先稳住：先把家庭沟通从“情绪对撞”变成“进度同步”。
2. 当前局面判断：属于“情绪稳定分支 + 行动计划分支”。
3. 现在最重要的事：建立每周可见进展。
4. 今天/本周行动清单：
- 今天：准备家庭沟通脚本。
- 本周：固定周报（投递、面试、计划）。
5. 可以说的话（对家人）：
“我已经在执行计划，这周重点是X，下周重点是Y，我会每周同步进度。”
6. 风险提醒：用争吵替代沟通会消耗执行力。
7. 时间线：7天建立沟通节奏，30天稳定推进，90天完成阶段目标。
8. 行动收束总结：让家人看到过程，压力会下降。
