# 示例：HR 施压签字（对话）

## 用户输入

“HR 说今天不签就没有 N+1，我该怎么回？”

## Skill 输出

1. 先稳住：先把“压力话术”转成“书面问题”。
2. 当前局面判断：属于“裁员流程分支”，核心风险是时限压力。
3. 现在最重要的事：争取延期并确认条款细节。
4. 今天/本周行动清单：
- 今天：发延期申请 + 条款确认清单。
- 本周：会后邮件纪要、必要时咨询律师。
5. 可以说的话：
“我需要核对条款，请将签署截止延至 [日期]，并书面确认补偿计算和附加义务。”
6. 风险提醒：口头承诺不等于有效条款。
7. 时间线：7天完成条款确认，30天完成求职节奏，90天确定稳定方向。
8. 行动收束总结：先拿书面再决定，不被当天节奏裹挟。
