# 示例：是否仲裁（对话）

## 用户输入

“我不确定要不要走劳动仲裁，怕耗时间也怕吃亏。”

## Skill 输出

1. 先稳住：先不急着做结论，先评估证据和争议点。
2. 当前局面判断：属于“裁员流程分支”，涉及法律风险。
3. 现在最重要的事：整理证据并咨询专业律师。
4. 今天/本周行动清单：
- 今天：做争议点清单和证据清单。
- 本周：做一次专业咨询，评估成本与收益。
5. 可以说的话（对 HR）：
“这部分我需要先做专业咨询，请先保留书面沟通记录。”
6. 风险提醒：没有证据基础直接硬上，成本高、成功率低。
7. 时间线：7天完成评估，30天推进主路径，90天复盘结果。
8. 行动收束总结：先做证据评估，再决定是否仲裁。
