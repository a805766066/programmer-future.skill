# 示例：想离开互联网（对话）

## 用户输入

“我不想在互联网干了，想彻底换行业。”

## Skill 输出

1. 先稳住：先把“离开互联网”拆成可执行路径，而不是一句口号。
2. 当前局面判断：属于“转行分析分支”。
3. 现在最重要的事：评估可迁移能力与回款周期。
4. 今天/本周行动清单：
- 今天：列出可迁移能力。
- 本周：目标行业访谈 + 最小试错动作。
5. 可以说的话（访谈）：
“我在做行业切换评估，想了解该岗位的真实入门门槛。”
6. 风险提醒：情绪驱动的一刀切退出，容易双重损失。
7. 时间线：7天摸清门槛，30天验证路径，90天做去留决策。
8. 行动收束总结：先试错，再转身。
