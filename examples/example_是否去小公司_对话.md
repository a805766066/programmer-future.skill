# 示例：是否去小公司（对话）

## 用户输入

“现在有个小公司 offer，我担心不稳定，要不要去？”

## Skill 输出

1. 先稳住（情绪稳定）
先别用“大/小公司”标签替代判断，先看经营和岗位真实情况。

2. 当前局面判断
你处于“方向决策分支”，核心是稳定性、职责边界和成长机会。

3. 现在最重要的事
做一轮小公司风险体检：现金流、客户稳定性、团队与汇报链。

4. 行动清单（今天 / 本周）
- 今天：确认业务来源、现金流周期、团队规模和直接汇报对象。
- 今天：确认试用期标准、转正条件、薪资发放规则。
- 本周：并行推进其他机会，避免单点押注。

5. 可以说的话（HR / 领导 / 家人 / 面试）
“我愿意考虑小团队机会，但希望先确认业务稳定性、岗位边界和绩效规则，确保双方预期一致。”

6. 风险提醒
不看业务基本面只看头衔，入职后风险高。

7. 接下来时间线（7天 / 30天 / 90天）
7天完成风险核查；30天评估团队与岗位匹配；90天判断是否长期投入。

8. 行动收束总结
先做尽调，再做选择，不凭想象入场。
