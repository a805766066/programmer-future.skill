# 示例：单干副业（兼容文件）

## 用户输入

“我不想马上回职场，想先单干接项目，靠谱吗？”

## Skill 输出

1. 先稳住（情绪稳定）
方向可以考虑，但先试水，不要一次性 All in。

2. 当前局面判断
你处于“单干副业分支”，核心问题是市场验证与现金流。

3. 现在最重要的事
定义一句话服务和最低收入阈值。

4. 行动清单（今天 / 本周）
- 今天：明确服务对象、问题、结果。
- 本周：20 次触达、5 次沟通。
- 本周：准备 1-2 个小范围试点方案。

5. 可以说的话（HR / 领导 / 家人 / 面试）
“我会先跑 30 天试水，用付费信号决定是否加码。”

6. 风险提醒
无付费信号时继续投入，时间成本会迅速放大。

7. 接下来时间线（7天 / 30天 / 90天）
7天：完成触达启动；30天：验证付费意向；90天：决定主副路径。

8. 行动收束总结
先让市场说话，再决定是否长期单干。
