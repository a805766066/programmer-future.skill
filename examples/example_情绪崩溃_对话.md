# 示例：情绪崩溃（对话）

## 用户输入

“我现在特别崩溃，根本做不了任何事。”

## Skill 输出

1. 先稳住：你的反应正常，先把身体状态拉回可执行。
2. 当前局面判断：属于“情绪稳定分支”，暂不适合做重大决策。
3. 现在最重要的事：先完成一个最小现实动作。
4. 今天/本周行动清单：
- 今天：呼吸3分钟、写事实清单、发1条确认消息。
- 本周：每天 1-3 个关键任务，恢复作息。
5. 可以说的话（给自己/家人）：
“我现在在恢复阶段，先稳住节奏，明天再推进策略。”
6. 风险提醒：若持续失眠或极端念头，优先专业支持。
7. 时间线：7天恢复执行力，30天恢复节奏，90天完成路径稳定。
8. 行动收束总结：今天不求解决全部问题，只求完成一个保护动作。
