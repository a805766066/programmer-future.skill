# 示例：找工作两个月没有 Offer（对话）

## 用户输入

“我投了两个月，面了不少，还是没 offer，怎么办？”

## Skill 输出

1. 先稳住：先停止自我否定，做结构化复盘。
2. 当前局面判断：属于“方向决策分支”，问题在策略而不一定在能力。
3. 现在最重要的事：定位“卡点在简历、面试还是渠道”。
4. 今天/本周行动清单：
- 今天：整理投递-面试数据。
- 本周：改一版简历叙事、补两类岗位、提升内推占比。
5. 可以说的话（找前同事内推）：
“我在看 [方向] 岗位，方便帮我看看是否有合适内推吗？”
6. 风险提醒：盲目加投递量但不改策略，收益很低。
7. 时间线：7天完成策略切换，30天看反馈，90天完成路径收敛。
8. 行动收束总结：用数据复盘替代情绪归因。
