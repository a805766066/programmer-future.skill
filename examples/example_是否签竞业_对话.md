# 示例：是否签竞业（对话）

## 用户输入

“HR 给了竞业协议，我不知道该不该签，怕影响后面找工作。”

## Skill 输出

1. 先稳住（情绪稳定）
先不要被“今天必须签”带节奏，先看清条款再决定。

2. 当前局面判断
你处于“裁员流程分支”，核心问题是竞业范围和补偿是否清晰。

3. 现在最重要的事
先拿到竞业条款的完整书面说明（范围、期限、补偿、违约责任）。

4. 行动清单（今天 / 本周）
- 今天：要求 HR 书面明确竞业四要素。
- 今天：对照你可能去的岗位判断冲突风险。
- 本周：若风险高，咨询专业劳动法律师。

5. 可以说的话（HR / 领导 / 家人 / 面试）
“请书面明确竞业适用范围、期限、补偿标准与发放安排。在信息完整前，我暂不做最终确认。”

6. 风险提醒
竞业条款不清就签，后续换工作会被动。

7. 接下来时间线（7天 / 30天 / 90天）
7天完成条款确认；30天完成求职路径避险；90天稳定新岗位方向。

8. 行动收束总结
先看清代价，再签字，不做情绪型决定。
