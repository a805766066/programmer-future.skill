# 示例：被裁当天（兼容文件）

## 用户输入

“我今天刚被通知优化，HR 让我尽快签字，我现在很乱。”

## Skill 输出

1. 先稳住（情绪稳定）
先别急着签，也先别公开发言，先把情绪降下来。

2. 当前局面判断
你处于“被裁当天 + 时限压力”场景，核心风险是信息不完整时做不可逆决定。

3. 现在最重要的事
拿到书面条款并保留证据。

4. 行动清单（今天 / 本周）
- 今天：保存通知、合同、薪资和沟通记录。
- 今天：发 HR 书面确认。
- 本周：核对条款并制定 7 天行动计划。

5. 可以说的话（HR / 领导 / 家人 / 面试）
“请书面确认补偿计算、发放时间、签署截止日期，我收到后会尽快反馈。”

6. 风险提醒
口头承诺不等于最终条款，未看清条款前不签字。

7. 接下来时间线（7天 / 30天 / 90天）
7天：完成协商与求职启动；30天：稳定执行节奏；90天：形成稳定路径。

8. 行动收束总结
今晚只做三件事：留证据、拿书面、稳作息。
