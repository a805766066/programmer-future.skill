# Release Notes v0.1

## 中文版

### 版本名称

v0.1 - 裁了吗（首个公开版本）

### 项目简介

`裁了吗` 是一个面向中国程序员的开源 Agent Skill 仓库。  
它不是网站或 SaaS，而是可被 Agent 调用的 Markdown 决策与行动技能。

### 当前版本包含的功能

- 统一 8 段输出结构
- 多分支决策分流（裁员、情绪、方向、转行、单干、计划）
- 可复制话术模板库（HR/领导/家人/面试）
- checklist 清单模板（当天/一周/30天/90天）
- 多场景中文示例对话

### Skill 模块列表

- `SKILL.md`
- `decision-tree.md`
- `workflows.md`
- `scripts.md`
- `scenarios.md`
- `persona.md`
- `boundaries.md`
- `resources/*.md`
- `examples/*.md`

### 示例场景

- 刚被裁当天
- HR 施压签字
- 情绪崩溃
- 找工作两个月没有 offer
- 是否签竞业 / 是否仲裁
- 是否降薪换工作 / 是否去小公司 / 是否出海远程

### 后续计划

- 增加更多城市与行业差异化案例
- 增加 offer 对比与入职风险检查模板
- 增加家庭压力与现金流协同决策模板
- 增加多 Agent 接入示例

### 项目定位

面向中国程序员职业危机阶段的 Agent Skill，强调“先稳住，再判断，再行动”。

## English Version

### Version

v0.1 - Laid Off Yet\? (Initial Public Release)

### Project Summary

`裁了吗` (Laid Off Yet?) is an open-source Chinese Agent Skill repository for software engineers facing career disruption.  
It is not a web app or SaaS, but a markdown-first skill system for practical decision and action support.

### What’s Included in v0.1

- A consistent 8-section response structure
- Decision routing for layoffs, emotion, direction, transition, freelance, and planning
- Copy-ready scripts for HR/manager/family/interview communication
- Checklist templates (day 1 / week 1 / 30-day / 90-day)
- Real-world Chinese scenario examples

### Skill Modules

- `SKILL.md`
- `decision-tree.md`
- `workflows.md`
- `scripts.md`
- `scenarios.md`
- `persona.md`
- `boundaries.md`
- `resources/*.md`
- `examples/*.md`

### Example Scenarios

- Layoff day response
- HR signing pressure
- Emotional breakdown
- Two months job search with no offer
- Non-compete / arbitration decisions
- Pay-cut decision / small-company decision / remote-overseas decision

### Next Steps

- Add more city and industry-specific cases
- Add offer comparison and pre-onboarding risk templates
- Add family-pressure + cashflow decision support templates
- Add more multi-agent integration examples

### Positioning

A practical Agent Skill for Chinese software engineers in career crisis, focused on: stabilize first, decide clearly, act next.
